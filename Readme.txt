Nome do Projeto: Site Mastro D'Ascia
Descrição: Site institucional do estaleiro Mastro D'Ascia de Florianópolis - SC, especializado em catamarãs e barcos de pesca.

Propósito: Informar sobre os produtos, promover a venda de catamarãs, fornecer informações técnicas e captar contatos e clientes.

Linguagem e Ferramentas Usadas:
- HTML, CSS, JavaScript para estrutura e estilo do site.
- PHP para envio de formulários e outras funcionalidades de back-end.

Estrutura de Pastas:
- /assets: imagens, arquivos CSS, JavaScript e documentos de download.
- /forms: formuários de mensagens.
- /portfolio: página dos produtos.

Instruções:
O site foi projetado para ser informativo e fácil de navegar, com links diretos para produtos e uma seção de artigos técnicos.

Contato:
Para suporte, sugestões ou dúvidas, entre em contato com Progenese Tecnologia.
